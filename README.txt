This repository contains demonstration applications for solving 
problems in computational economics.  The initial demos are
intended for use at the Initiative on Computational Economics
(ice.uchicago.edu) and ECON 41701 at the University of Chicago.

Demos:

  ols           : demonstrates how to use a cluster to show OLS is consistent
                  by generating synthetic data and estimating it for 
                  10, 100, 100, ... observations (N).
